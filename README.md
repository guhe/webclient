# webclient

## 功能
* 显示组合基本信息
* 显示组合持仓明细
* 添加、查看交易信息（卖、卖）
* 添加、查看份额信息（申购、赎回）
* 计算历史每日单位净值
* 显示组合净值曲线
* 支持沪深B股
* 修改组合现金余额
